# Get All Player Statuses

Retrieve injury status information for all NFL players in a simplified format focused on status and injury data.

<RequestExample>
```bash
curl -X GET https://ifpwy9pbk0.execute-api.us-east-1.amazonaws.com/prod/player
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message": "All player statuses retrieved successfully",
  "data": [
    {
      "player_name": "Kadarius Toney",
      "status": "Ready for Play",
      "reported_date": "2024-09-10",
      "injury": "Ankle",
      "team": null,
      "position": "WR",
      "injury_score": "4"
    },
    {
      "player_name": "C.J. Stroud",
      "status": "Ready for Play",
      "reported_date": "2025-01-14",
      "injury": "Undisclosed",
      "team": null,
      "position": "QB",
      "injury_score": "0"
    }
  ]
}
```
</ResponseExample>

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `success` | boolean | Request success status |
| `message` | string | Response message |
| `data` | array | Array of player status objects |

### Player Status Object Fields

| Field | Type | Description |
|-------|------|-------------|
| `player_name` | string | Player's full name |
| `status` | string | Current injury status |
| `reported_date` | string | Date injury was reported (YYYY-MM-DD) |
| `injury` | string | Type of injury |
| `team` | string | Player's team (may be null) |
| `position` | string | Player's position |
| `injury_score` | string | Injury severity score (0-157) |

## Differences from Get All Players

This endpoint provides a more focused response compared to the "Get All Players" endpoint:

- **Simplified data**: Only essential status information
- **Consistent format**: All players have the same status object structure
- **Faster response**: Less data transfer
- **Status-focused**: Optimized for injury status tracking

## Use Cases

- **Fantasy Football Apps**: Quick status overview for all players
- **Injury Tracking**: Monitor all players' injury status
- **Dashboard Widgets**: Display status summaries
- **Bulk Analysis**: Process injury patterns across the league
