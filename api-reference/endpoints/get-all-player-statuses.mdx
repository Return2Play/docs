# Get All Player Statuses (Today)

Returns today\'s predictions from `PLAYER_PREDICTION_FOR_DAY` as simple OUT/ACTIVE statuses (v1) for every player.

<RequestExample>
```bash
curl -X GET \
  -H "Authorization: Bearer <JWT>" \
  https://ifpwy9pbk0.execute-api.us-east-1.amazonaws.com/prod/player
```
</RequestExample>

<ResponseExample>
```json
[
  {
    "player_id": "12831",
    "player_name": "Tyrod Taylor",
    "status": "OUT",
    "update_date": "2025-08-13"
  },
  {
    "player_id": "14688",
    "player_name": "Justin Tucker",
    "status": "ACTIVE",
    "update_date": "2025-08-13"
  }
]
```
</ResponseExample>

## Response Fields

Array of objects with:

| Field | Type | Description |
|-------|------|-------------|
| `player_id` | string | SportsData player ID |
| `player_name` | string | Player name |
| `status` | string | V1 status, `OUT` or `ACTIVE` |
| `update_date` | string | Date (YYYY-MM-DD) the prediction applies to |

## Notes

- Requires Authorization header with a valid JWT.
- Data comes from the daily predictions table `PLAYER_PREDICTION_FOR_DAY` populated by the final report Lambda.

## Use Cases

- **Fantasy Football Apps**: Quick status overview for all players
- **Injury Tracking**: Monitor all players' injury status
- **Dashboard Widgets**: Display status summaries
- **Bulk Analysis**: Process injury patterns across the league
